--Task List Tool-- 

#Una aplicación interactiva en consola creada en C++ para gestionar tareas diarias.#

º__Características__º
***Agregar Tareas: Inserta tareas dinámicamente en memoria utilizando vectores de C++ (`std::vector`).***
***Ver Tareas: Muestra la lista de tareas numerada con un contador dinámico.***
***Eliminar Tareas: Elimina tareas con validación de rangos para evitar errores de memoria.***
*** Menú Interactivo: Menú repetitivo usando control de flujo limpio (`do-while` y `switch`).***

 hecho en:
* Lenguaje: C++
  Librerías Estándar: `<iostream>`, `<vector>`, `<string>`

¿Cómo Ejecutarlo?

1. Compila el archivo en tu terminal:
   ```bash
   g++ -o task_tool main.cpp
